Attention meets Power Spectrum

Appendix C

In Section 5.1 of our article we show that there is a strong correlation between our attention features and certain features derived from power spectrum of the audio sample (we will call them "spectral features") that are commonly used in speech analysis. Here we present the full results for Pearson Correlation coefficient between Spectral features and all 6 of our attention features, namely:

  • [Mean element of the] Trace of the attention matrix
  • Asymmetry measure of the attention matrix (sum of the elements above main diagonal)
  • [Average] Attention to the next element
  • [Average] Attention to the previous element
  • [Average] Length of the bars for H0-barcode when attention matrix considered a graph (per described in the paper)
  • [Average] Length of the bars for H0-barcode when attention matrix considered a point cloud (distances between tokens are calculated as l1-distances between rows of the matrix)
